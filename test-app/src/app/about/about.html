About TestBites

TestBites is a simple demo webshop used to demonstrate automated end-to-end testing workflows.

TestBites is a small demo webshop created for testing purposes. The application simulates a simple online food ordering system where users can browse products, add items to a cart, and complete a purchase.

This project was developed as part of a Bachelor’s thesis in Information Technology to demonstrate and compare end-to-end (E2E) testing tools such as Playwright and Cypress. The store provides realistic user flows that can be automated and tested to evaluate the strengths and differences between the two frameworks.

The application is built with Angular and includes typical ecommerce features such as product listings, cart management, filtering, and checkout simulation. These interactions make it suitable for automated end-to-end testing scenarios.

Project Purpose

This demo application was created to support a Bachelor’s thesis that compares end-to-end testing tools Playwright and Cypress. The system simulates a small online food store and provides typical ecommerce user flows that can be automated and tested.

Features

  • Browse food products
  • Filter products by category
  • Search products
  • Add and remove items from the cart
  • Adjust item quantities
  • Simulated checkout process

Testing Focus

The application includes common user interactions found in ecommerce websites, such as product browsing, cart management, and checkout confirmation. These interactions allow automated end-to-end tests to validate the user interface and application behavior across multiple scenarios.
